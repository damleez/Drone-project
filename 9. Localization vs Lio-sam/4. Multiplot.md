MULTI PLOT
===

### 1. Setting
```
sudo apt-get install ros-noetic-rqt-multiplot
```
- 설치 후, CMD
```
rqt_multiplot
```

#### - 순서
1) Topic 명 추가
2) lio_sam/mapping/odometry 선택하고 pose x, y 선택
3) 마찬가지로 odometry/filtered 선택하고 pose x, y 선택
4) 실행 버튼 누르면 current position 찍힘
